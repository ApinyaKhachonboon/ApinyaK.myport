Ms-Paint (FinalMsPaint)

Class diagram

ใช้การเขียนแบบ OOP
Class MS_PAINT_GUI ทำหน้าที่แสดงผล เครื่องมือต่างๆของโปรแกรม ดังต่อไปนี้

- ปุ่ม File ใช้ JMenu ในการสร้าง เมื่อ click จะมีปุ่ม save และ load แสดงขึ้น

- Save ทำหน้าที่ save จาก Object BlankArea ทุกพิเซลในรูปแบบของไฟล์ นามสกุลรูปภาพเช่น PNG, JPG, GIF

- Load ทำหน้าที่ Load ไฟล์ นามสกุลรูปภาพ เช่น PNG, JPG, GIF มาแสดงผลใน Object BlankArea

- Group ของ Brash และ Eraser จะมีขนาดให้ปรับในรูปแบบ Dropdown สามารถเลือกได้ตามต้องการ

- ปุ่ม undo สามารถกด เพื่อย้อนขั้นตอนทีละขั้น สามารถใช้ Short-cut Ctrl+z แทนได้

- ปุ่ม Rotate ทั้ง 3 ปุ่มใช้ในการหมุน Object Shape ด้านขวา

- ปุ่ม Triangle, Rect, Oval และ Star ใช้ในการสร้างรูปร่าง ตามชื่อปุ่มที่ว่าไว้ สามารถใช้ Mouse Drag จนกว่าจะได้ขนาดที่ต้องการได้ ซึ่งจะไปเรียกใช้ Object Shapes

- ปุ่ม Color 1, Color 2 ใช้เลือกสีที่ต้องการ ซึ่งใช้ร่วมกันกับทุก Object ที่ใช้สี โดยทำงานจาก Object ชื่อ ColorPicker และ ColorBtn

Class BlankArea

- เมื่อกดใช้ Brash หรือ Eraser แล้วนำ Mouse Drag ลงบน BlankArea จะมีการเก็บ Data โดยใช้ List ของ Java เป็นตัวเก็บพิกัดที่ลากพร้อมกับทำการวาดเส้นลงบน BlankArea ตามพิกัดเพื่อให้เส้นที่ลากต่อเนื่องกัน จนกระทั่งปล่อย จะถือเป็น 1 Object

- เมื่อกดใช้ Triangle, Rect, Oval, Star จะสร้าง Object ขึ้นมาตามที่เลือก ปรากฏขึ้นมาพร้อมกับ Object SelectRect ซึ่งจะเป็นกรอบของรูปร่างที่เลือก และจะถูกเก็บไว้ใน Data List ชื่อ allElements และ กด Click ด้านนอก Shape จะทำให้ SelectRect หายไป

Class SelectRect

- เมื่อถูกเรียกใช้จะสร้างกรอบเส้นปะสีน้ำเงินรูปสีเหลี่ยม โดยดูจากพิกัด Mouse ที่อยู่ใน BlankAera จะทำงานร่วมกับ Object ResizeRect เป็นสีเหลี่ยมเล็กๆ ขนาด 10px เพื่อให้ง่ายต่อการปรับรูปทรง

Class Shapes

- เมื่อถูกเรียกใช้จะรับค่า String มาเพื่อวาดรูปภาพ และ สามารถ Rotate ได้ตามการคำนวนทางคณิตศาสตร์ ทำงานตาม Method ด้านใน

Class ColorPicker

- สร้างมาจาก JDialog และด้านในเป็น JColorChooser

Class ColorPickBtn

- ไว้สำหรับเรียก ColorPicker อีกทีแล้วแสดงสีที่เลือกออกมา

Class ReSizeRect

- เป็นสี่เหลี่ยมเล็กๆขนาด 10 px ไว้สำหรับปรับขนาดของ Shape

Class Brush

- ไว้สำหรับปรับขนาดและสีของพู่กันรูปทรงเป็นวงกลม

Class Eraser

- ทำหน้าที่เหมือนยางลบไว้ลบสิ่งที่ไม่ต้องการออกจากหน้าของ BlankArea สามารถเปลี่ยนสีได้ตามต้องการ
