เกมฝึกทักษะการพิมพ์ (EV10)

Class diagram

Class EV10

- เมื่อถูกเรียกจะแสดงหน้า GUI ประกอบด้วย Time, Score, text field และสร้าง ลูกบอลที่มีข้อความด้านในขึ้นมาโดนข้อความด้านในจะนำมาจาก .TXT ไฟล์ จะทำการ Random สี ขนาด ตำแหน่ง ของลูกบอลมา 3 ลูก ทุกครั้งที่มีการกด Enter ที่ text field โปรแกรมจะทำการตรวจสอบว่าตรงกับที่มีในลูกบอลหรือ ไม่ถ้าตรง ลูกบอลจะถูกทำลายและจะได้ Score เพิ่ม พร้อมทั้งลูกบอลจะเกิดใหม่พร้อมข้อความถัดไป random ขนาด สี และ ตำแหน่งต่อไปทำไปจนกระทั่งหมดเวลา ในกรณีที่ลูกบอลตกถึงพื้นก่อน score จะ -1 และจะเกิดใหม่เช่นกัน

Class Bubble

- ด้านในจะมี Attribute ที่เกี่ยวกับ สี ข้อความ ตำแหน่งของ ลูกบอล และ Method move() ไว้ใช้ในการเคลื่อนที่ ถูกสร้างขึ้นเพื่อให้ Class EV10 นั้นเรียกใช้
