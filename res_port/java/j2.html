WordPad (FinalWordPad)

Class diagram

ใช้รูปแบบการเขียนแบบ OOP
Class WordPadGUI

- ปุ่มทุกปุ่มด้านซ้ายทั้งหมด จะส่งค่าตัวแปรรูปแบบเป็น SimpleAttributeSet เพื่อไปจัดการปรับค่าในแต่ละตัวอักษรใน JTextPane เนื่องจากใน JTextPane ไม่สารารถ Warp Text ได้จึงต้องเพิ่ม Class WrapEditorKit ซึ่งสร้างมาจาก StyleEditorKit มาเพื่อจัดการเรื่องนี้

- ปุ่มด้านขวาจะจัดการเรื่องตำแหน่งของตัวอักษร ซึ่งจะแบ่งไปแต่ละ Paragraph จะแบ่งโดย Text ด้านใน JTextPane ด้วย “\n” เพื่อให้ผู้ใช้ได้ใช้งานอย่างอิสระ และ มี Object Ruler ที่ช่วยควบคุบตำแหน่งของแต่ละ Paragraph ด้วย

- ปุ่ม undo ทำงานตาม Object UndoManager ซึ่งจะดูแลเรื่องจากย้อนขั้นตอนการทำของ JTextPanel สามารถใช้ Shortcut Ctrl+z แทนได้

- ปุ่ม save ทำหน้าที่ Save ข้อมูลใน JTextPane ทั้งหมดในรูปแบบไฟล์สกุล .TFF

- ปุ่ม load ทำหน้าที่ Load ข้อมูลจากไฟล์สกุล .TFF มาแสดงผลบน JTextPane

Class WarpEditorKit

- ทำหน้าที่ warp text ให้ถูกต้องตรงกับขนาดที่ต้องการของแต่ละ paragraph ซึ่งจะไปเรียกใช้ WrapColumnFactory อีกที

Class WrapColumnFactory

- จะเช็คว่า text ที่อยู่ใน JTextPane อยู่ในตำแหน่งไหนและจบ paragraph หรือยังถ้าถึงขอบเขตที่กำหนดจะทำการขึ้นบรรทัดใหม่เอง

Class WarpLebelView

- เอาไว้ไม่ให้ text แสดงผลผิดตำแหน่งจะทำให้ text บัคได้

Class FontCellRenderer

- เช็ค Font แต่ละ cell ให้แสดงในตำแหน่งที่ถูกต้อง

Class MyParagraphView

- ใช้สำหรับปรับทุก Attribute ใน paragraph นั้นๆ และจะคืนค่าตามตำแหน่งของ Curser จะทำงานร่วมกับ Ruler ด้วย

Class Ruler

- ในการควบคุมตำแหน่งของแต่ละ Paragraph นั้นจะทำงานตาม ตำแหน่งของ Curser ของแต่ละ Paragraph ซึ่งจะรับค่าของเดิมเพื่อย้าย Ruler ให้ตรงกับค่าของแต่ละ paragraph ได้จำไว้ด้วยด้านใน จะมี Shape รูปสามเหลี่ยมเพื่อให้ผู้ใช้เลื่อนได้สะดวก

Class TriangleShape

- ทำหน้าที่แสดงรูปสามเหลี่ยมสีดำบน Ruler เพื่อไว้จัด Box view ของแต่ละ Paragraph

Class ColorPickerBtn

- ใช้กำหนดค่าสี และ Highlight ของตัวอักษรซึ่งรับ Parameter มาจาก JColorChooser เพื่อแสดงค่าสีที่เลือกให้เห็นอีกที
