เกมกระต่ายกระโดด (Ev12)

Class diagram

Class EV12

- ใช้แสดงผลภาพทั้ง score และ Class starBG, Rabbit, Bell เพื่อให้เกมดำเนินไปจนจบเกมจะมี Method jump() ในการควบคุมการกระโดดของกระต่าย

Class Rabbit

- จะมี Method ไว้ใช้ควบคุมการเคลื่อนไหว มี Method get และ set ไว้ระบุตำแหน่งและ ปรับตำแหน่งตามการเคลื่อนไหวของ mouse

Class Bell

- Bell หรือ กระดิ่ง จะทำหน้าที่ร่วงลงมาเรื่อยๆ จนกว่าจะถึงพื้นหรือ สัมผัสโดนตัวกระต่าย ซึ่งเมื่อกระดิ่งตกหรือโดนตัวกระต่ายแล้วกระดิ่งจะ เกิดใหม่ด้านบนและ random ตำแหน่งการเกิด

Class CheckHit

- Class Bell จะทำการเรียก Class นี้เพื่อตรวจสอบการชนโดนใช้สูตรของพิทากอรัส
