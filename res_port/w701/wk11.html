ทดลองทำ web site ซื้อ ขาย อสังหา ในส่วนแรกจะมี animation ทำให้ div หมุน 1 รอบโดยใช้ anime.js ส่วนด้านบนเมนูสร้างโดย div>ul>li และจัด float left-right ต่อมาส่วนของการ search ใช้เป็น form เมื่อกด click search จะนำค่าจาก input text ไปค้นหาข้อมูลที่ต้องการหา โดยจะใช้ ajax จึงไม่จำเป็นต้องโหลดหน้า web ใหม่โดยทำการ get json จาก url อื่นโดยใช้ jquery จากนั้นนำข้อมูลที่ได้ ใส่ใน template และนำไป append ในอีก div

ภาพด้านบนคือผลลัพธิ์ที่ได้และนอกจากนี้ยังมีลักษณะ responsive อีกด้วยและได้ใช้ anime.js เพื่อใช้การแสดง animation เมื่อ scroll ถึงในตำแหน่งที่ div แสดง tile ที่ search นอกจากนี้ยังสามารถกดที่แสดงรายละเอียดเพื่อ show ข้อมูลในแต่ละ tile พร้อมด้วยแสดง google map ด้านล่างด้วยโดยเมื่อ click ที่แสดงรายละเอียดจะเอาข้อมูลจาก tile ใส่ไว้ในค่า lat, lng ของ google map

รูปด้านบนคือผลลัพธิ์ที่ได้ ซึ่งจะเห็นว่ามีข้อมูลจาก tile ที่กดและยังแสดงภาพด้านในด้วย นอกจากนี้เมื่อลองเช็คสถาที่ในgoogle map แล้วปรากฏว่าถูกต้อง
