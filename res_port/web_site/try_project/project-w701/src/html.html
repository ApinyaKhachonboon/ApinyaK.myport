HTML

HTML (Hypertext Markup Language) คือภาษาสำคัญในการสร้าง Web Site ซึ่งจะแสดงผลผ่าน Web Browser ในตัว HTML นั้นจะประกอบด้วย tag หลายรูปแบบให้เลือกใช้มากมายให้เหมาะกับความต้องการ ในที่นี้จะยกตัวอย่าง tag ที่ใช้บ่อยพร้อมคำอธิบายบางส่วน

รูปด้านบนจะเป็นรูปแบบมาตรฐานของ HTML5 ซึ่งจะมี tag html และด้านในประกอบไปด้วย head ซึ่งเอาไว้ใส่เกี่ยวกับการคำนวณเช่นใส่ tag script สำหรับ JavaScript หรือ CSS ซึ่งเป็นภาษาเอาไว้จัดแต่งรูปร่างความสวยงามของแต่ละ Element ต่อมาเป็น body ด้านในนี้จะเอาไว้ใส่ tag ที่เอาไว้แสดงผลบนเว็บ browser ซึ่ง tag ที่ใช้บ่อยๆโดยทั่วไปนั้นมี

Tag h1 – h6 คือ tag ที่ไว้แสดงหัวข้อต่างจะมีขนาดตั้งแต่ 1 – 6

ต่อมาเป็น tag form คือ tag ที่เอาไว้เก็บข้อมูลต่างๆ เพื่อเอาไว้ส่งแบบฟอร์มไปยังเครื่อง server เพื่อทำการเก็บข้อมูลอีกทีซึ่งสามารถเซ็ต action หรือรูปแบบการส่งได้ 2 แบบคือ GET และ POST โดยทั่วไปจะนิยมการส่งแบบ POST มากกว่าเพราะปลอดภัยและสามารถส่งจำนวนข้อมูลได้มากกว่าแบบ GET ซึ่งด้านใน tag form เราสามรถใส่ tag input ต่างๆเพื่อให้ผู้ใส่กรอกข้อมูลที่เราต้องการใส่ลงไปได้ซึ่งจะมี type คล่าวๆดังนี้

  • text คือ กล่องที่ไว้เก็บข้อความต่างๆ
  • checkbox คือ กล่องตัวเลือกจะมีเครื่องหมายถูกหน้า value ที่ผู้ใช้เลือก
  • color จะปรากฏปุ่มเมื่อคลิ๊กจะสามารถเลือกสีตามต้องการได้ซึ่งจะคืนค่าเป็นสีที่เลือก
  • email คือกล่องที่ไว้เก็บข้อความที่เป็นในลักษณะของ email
  • password คือกล่องที่ไว้เก็บข้อความแต่จะไม่ปรากฏข้อความที่พิมพ์ลงไปให้เห็น
  • image ไว้แสดงรูปภาพตาม src ที่กำหนด
