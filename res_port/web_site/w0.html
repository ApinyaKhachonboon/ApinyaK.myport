Web-game (Bomberman)

Class Diagram

game.html

เป็นไฟล์ main หลักของเกม ซึ่งจะทำการ Import Class Block, Bomb, BreakBlock, Enemy, Item, Player อีกทีเพื่อนำแต่ละ Object มาวาดลงใน canvas ด้วย Method เช่น createBlock, createBreakBlock ใช้วาดลงในตำแหน่งที่ต้องการ เพื่อใช้ในการ Genarate map ขึ้นมา นอกจากนี้ยังมี Method keyDownHandler ใช้ในการดักคีย์ที่ต้องการใช้ในการเล่นเกม นอกจากนี้ยังมี method ต่างๆที่เกี่ยวกับกลไกของเกม เช่น canPlant ทำหน้าที่เช็คว่าระเบิดเกินจำนวนที่กำหนดหรือยัง และคำนวณให้วางตรงกับช่องพอดีด้วย สุดท้าย Method draw ทำหน้าที่ update แต่ละ frame เพื่อให้เกิด animation ขึ้นมาโดยมี Method built in ที่เรียกตัวเองซ้ำๆชื่อ requestAnimationFrame

player.js

ทำหน้าที่เหมือนกับตัวผู้เล่นโดยเก็บค่าตำแหน่ง, รูป, score, bomb, blaze และยังมี Method เช็คตำแหน่งต่างๆว่าสามารถเดินไปได้หรือไม่ มีการตรวจสอบการชนของ Enemy, Item และมี Method Reborn เวลาผู้เล่นตายจะไปเกิดในตำแหน่งเดิม

Enemy.js

จะมีทุกอย่างที่คล้ายกับ Class Player จะมีที่แตกต่างคือ จะมี Method ในการ random ทิศทางที่จะเดินเมื่อเดินไปติด block จะเรียก random ทิศทางใหม่ทุกครั้ง

bomb.js

ทำหน้าที่แทนระเบิดที่ผู้เล่นได้วางไว้ นอกจากนั้นยังมี Method checkBlock เพื่อดูว่า bomb ถูกวางในตำแหน่งที่ถูกต้องหรือไม่ และทำการ random Item จาก breakblock ที่ถูกทำลายไปด้วย และสุดท้ายคือ Method explode จะทำการสร้าง fires ตามเงื่อนไขที่กำหนดไว้ในทิศทางที่ถูกต้อง

fires.js

ทำหน้าที่เป็นเหมือนไฟของระเบิดซึ่งด้านในจะเป็นตำแหน่ง, รูปในทิศทาง ต่างๆ ซึ่งจะถูกเรียกโดย Method explode จาก Class Bomb

breakblock.js

ทำหน้าที่เหมือนสิ่งกีดกวาง สามารถถูกทำลายได้เมื่อเกิด explode จาก Class Bomb และมี Method setDoor เมื่อระถูกทำลายจะส้รางประตูขึ้นมา

block.js

มีทุกอย่างที่เหมือนกับ breakblock แต่ไม่สามารถถูกทำลายได้
